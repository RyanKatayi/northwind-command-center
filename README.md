# Northwind Command Center

Internal operations tool for Northwind Coffee, a (fictional) wholesale coffee roaster. Built for the Northwind OS challenge (see [CHALLENGE.md](./CHALLENGE.md)). Work in progress.

## Run

Needs Node 20+.

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

The mock data in `data/` is the read-only source.
